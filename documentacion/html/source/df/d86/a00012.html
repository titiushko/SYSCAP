validar_parametros (   $codigo_departamento,
  $codigo_municipio = FALSE 
)
private

Método que comprueba que sean válidos los parámetros que se resiben de la URI El método devuelve TRUE si son parámetros válidos, de lo contrario el método devuelve FALSE El método comprueba las siguientes validaciones:

  • código del departamento:
    • no debe de ser nulo o vacío
    • debe de ser un número entre el 1 y el 14
  • código del municipio:
    • no debe de ser nulo o vacío
    • debe de ser un número entre el 1 y el 262
    • debe de pertenecer al código del departamento correcto
